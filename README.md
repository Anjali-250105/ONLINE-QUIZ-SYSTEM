ONLINE QUIZ SYSTEM
PROJECT OVERVIEW

The Online Quiz System is a web-based application used to conduct quizzes and assessments online.
It helps students test their knowledge and provides instant results.
The system reduces manual work and saves time.

OBJECTIVES

• To create a simple and user-friendly quiz system
• To provide instant feedback to users
• To automate the quiz process
• To make learning interactive
PROBLEM STATEMENT

Traditional quiz systems are time-consuming and difficult to manage.
Manual evaluation causes delays and errors.
An online automated system is needed for efficient quiz management.

TECHNOLOGIES USED

• HTML
• CSS
• JavaScript

FEATURES

• Interactive quiz interface
• Multiple-choice questions
• Instant score calculation
• Start and restart quiz option
• Simple and attractive design
PROJECT STRUCTURE

• onlinequiz.html
• README.md

HOW TO RUN THE PROJECT

Download the project files

Open onlinequiz.html in any web browser

Click on START QUIZ
